# Zomato Restaurant Analytics Dashboard

A professional Business Intelligence project built using Microsoft Power BI and a Star Schema data model to analyze Zomato restaurant data.

This project transforms raw restaurant data into an interactive analytics dashboard that provides insights into restaurant performance, customer preferences, ratings, cuisines, pricing trends, and geographical distribution.

---

## Project Overview

The objective of this project is to design an end-to-end data analytics solution by:

- Cleaning and structuring raw data
- Building a scalable Star Schema data model
- Creating DAX-based analytical measures
- Developing interactive Power BI dashboards
- Extracting meaningful business insights

The dashboard helps stakeholders understand market trends, restaurant performance, customer engagement, and location-based opportunities.
